## Requirement 

- <a href="https://python.org/downloads">Python 3.6 or Latest Version.</a>
- Internet Connection
- Audio Speakers
- 10gb Drive Storage
- 8GB Ram (4GB Ram minimum)

**use this command on termial in JARVIS directory to install required packages.**
```python
pip install -r setup.txt
```
